KURZANLEITUNG
═════════════

  1. INSTALL.bat        → Einmalig: Installiert alles
  2. START.bat          → Startet den Sprachcomputer
                          (Browser öffnet sich automatisch)
  3. Im Browser:
     - Leertaste/Enter  = Signal geben
     - F11              = Vollbild (empfohlen!)
     - Die Optionen werden nacheinander hervorgehoben
     - Bei Signal wird die aktuelle Option gewählt

VORAUSSETZUNGEN
═══════════════

  • Windows 10/11
  • Python 3.10+ (https://www.python.org/downloads/)
    → Bei Installation "Add Python to PATH" ankreuzen!
  • Webcam (USB oder eingebaut) — für Kamera-Modus
  • Lautsprecher (für Sprachausgabe)
  • Browser (Chrome/Edge empfohlen)

KOMMUNIKATIONS-MODI
════════════════════

  HAUPTMENÜ
    ├── Schnell-Phrasen  (36 vorgefertigte Sätze)
    │     Bedürfnisse, Schmerzen, Lage, Soziales, Hilfe
    │
    ├── Buchstabieren   (Frequenz-optimiertes Layout)
    │     Mit Wortvorschlägen (500+ deutsche Wörter)
    │
    ├── Ja / Nein / Passe
    │     Für direkte Fragen
    │
    └── HILFE!          (Notruf-Alarm)
          3× schnelles Signal = Alarm aus jedem Modus

SCANNING-PRINZIP
════════════════

  1. ZEILEN werden nacheinander hervorgehoben (gelb)
  2. Signal → Zeile wird fixiert
  3. SPALTEN werden einzeln hervorgehoben
  4. Signal → Item wird ausgewählt
  5. 2,5 Sekunden CANCEL-COUNTDOWN
     (nochmal Signal = Abbruch → Sicherheit!)
  6. Sprachausgabe bestätigt die Auswahl

ALLE BAT-DATEIEN
════════════════

  INSTALL.bat       Erstinstallation
  SELFTEST.bat      Prüft ob alles funktioniert
  START.bat         ★ Kommunikation starten (Hauptprogramm)
  KALIBRIEREN.bat   Kalibrierung mit der Person
  AUFNEHMEN.bat     Videos aufnehmen
  ANNOTIEREN.bat    Aufnahmen annotieren
  VALIDIEREN.bat    Detektor-Qualität prüfen
  UPDATE.bat        Neueste Version holen

TIPPS
═════

  • Für den ersten Test: START.bat → Option 1 "Nur mit Tastatur"
    (funktioniert ohne Kamera, Leertaste = Signal)

  • F11 im Browser = Vollbild (empfohlen für den Patienten!)

  • Scan-Geschwindigkeit ist einstellbar beim Start
    Langsamer = sicherer, Schneller = effizienter

  • Bei Problemen: SELFTEST.bat zeigt was fehlt

  • Für Updates: UPDATE.bat oder INSTALL.bat erneut ausführen

ARCHITEKTUR
═══════════

  Python-Server (localhost:8000)
    ├── Kamera + Gesichtserkennung (MediaPipe)
    ├── Detektor (Schmitt-Trigger)
    ├── Scanning-Engine + TTS
    └── WebSocket → Browser

  React-Frontend (im Browser)
    ├── Scanning-Raster (große Buttons, Animationen)
    ├── Text-Anzeige + Wortvorschläge
    └── Empfängt Signale per WebSocket
